# ⚾️ AI와 함께하는 숫자 야구 게임 배포 프로젝트

이 프로젝트는 **VS Code + GitHub Copilot**을 활용한 바이브 코딩 실습 과제입니다. 
AI와 대화하며 게임을 완성하고, 깃허브 페이지를 통해 전 세계에 배포해보세요!

## 🎯 프로젝트 목표
1. **숫자 야구 로직 구현**: 3스트라이크를 맞추는 게임 기능을 만듭니다.
2. **웹 UI 디자인**: HTML/CSS를 활용해 세련된 게임 화면을 구성합니다.
3. **랭킹 시스템 연동**: Google Apps Script(GAS)를 이용해 결과를 구글 스프레드시트에 저장합니다.
4. **자동 채점 통과**: 지정된 함수 형식을 지켜 과제를 제출합니다.

---

## ⚠️ 자동 채점을 위한 필수 규칙 (필독!)

본 과제는 제출 시 자동으로 로직을 검사합니다. **자동 채점(Autograding)을 통과하기 위해 `script.js`에 아래 함수를 반드시 포함해야 합니다.**

### 1. 함수 이름 및 형식
- **함수명**: `checkAnswer(target, input)`
- **매개변수**:
    - `target`: 컴퓨터가 생성한 정답 세자리 숫자 (예: 234)
    - `input`: 사용자가 입력한 세자리 숫자 (예: 314)
- **반환값(Return)**: 스트라이크와 볼의 개수를 담은 객체
    - 예시: `{ strike: 1, ball: 1 }`

> **💡 코파일럿에게 이렇게 시켜보세요:**
> "숫자 야구 로직을 짤 때, 정답 세자리 숫자와 입력 세자리 숫자를 비교해서 `{strike: n, ball: m}` 객체를 반환하는 `checkAnswer` 함수를 반드시 포함해서 만들어줘."

---

## 🛠 파일 역할
- `index.html`: 게임의 구조 (UI)
- `style.css`: 게임의 디자인 (CSS)
- `script.js`: 게임 로직 및 GAS 연동 (JavaScript)
- `google_script.js`: 구글 스프레드시트에 붙여넣을 서버 코드

## 🚀 시작하기
1. 오른쪽 하단 **Live Server**를 클릭해 미리보기 화면을 켭니다.
2. `Ctrl + I`를 눌러 코파일럿에게 UI 제작을 먼저 요청하세요.
3. `checkAnswer` 함수 규칙을 지켜서 로직을 완성하세요.
4. 모든 기능이 작동하면 구글 스프레드시트 배포 후 URL을 연결하세요.
